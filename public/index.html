Modern & Intuitive GUI

By strictly following the macOS Human Interface Guideline and focusing
on a thorough UI/UX, Proxyman is intuitive and friendly to all users.

Native, High-performance, Full-featured

Exclusively built as a native app, Proxyman is a superb tool that helps
developers to complete their tasks and collaborate with the team.

Sophisticated Proxy Core

Proxyman is entirely written in Swift-NIO, Swift, Objective-C, C, C++, and designed to run uninterruptedly at high performance with minimum system resource consumption.

Thus, it optimizes RAM usage, battery life while maintaining the stability of your internet connection.

SSL Proxying

Proxyman acts as a man-in-the-middle that logs all HTTP/HTTPS traffic between your applications and SSL Web Server. Proxyman will dynamically generate certificates on-the-fly, which enables developers to see HTTPS in plain text.

Proxyman is also integrated with iOS simulators and iOS and Android devices.

Advanced debugging tools

Proxyman provides a full set of built-in tools that facilitate debugging procedures.

With this package analyzer, it would be much easier to observe and manipulate HTTPs requests/responses, or replicate bugs and collaborate with other team members.
